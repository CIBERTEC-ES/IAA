**Pasos para descargar e Instalar Java**

_En este momento, la última versión estable de Java es "Java 25", que a la fecha es una versión de soporte a largo plazo (LTS)._

   1\. Descargar Java de Oracle

> Info: https://www.oracle.com/java/technologies/downloads/#jdk25-windows
>
> Descargar: https://download.oracle.com/java/25/latest/jdk-25_windows-x64_bin.exe

2. Proceder con la instalación del ejecutable descargado
3. Abrir una ventana de CMD en windows y escribir el comando

> java --version

4. Si muestra resultado indicando la verisón de Java, entonces todo está instalado
